GoodLink 夸克 Cookie 抓取器

完整 Cookie 字串（__puus + _UP_* + tfstk + 全部）

需要开启无痕模式权限

  1. 打开 chrome://extensions
  2. 找到「GoodLink 夸克 Cookie 抓取器」
  3. 点击「详细信息」
  4. 开启「允许在无痕模式下运行」

导出设置

操作

  1. 打开无痕窗口，进入夸克网盘
  2. 在页面上登录（手机号/扫码）
  3. 登录成功后自动抓取全部 Cookie
  4. 导出 JSON 文件（含完整 cookie_string）
